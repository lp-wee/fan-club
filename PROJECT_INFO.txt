================================================================================
                       TROUBLESHOOTING
================================================================================

DATABASE CONNECTION ERROR:
$ psql postgres -c "SELECT version();"  # Check PostgreSQL
$ dropdb jobportal && createdb jobportal  # Recreate
$ psql jobportal -f scripts/init-db.sql  # Reinit schema

PORT ALREADY IN USE:
$ lsof -ti:3000 | xargs kill -9  # Kill frontend
$ lsof -ti:3001 | xargs kill -9  # Kill backend

DEPENDENCIES ISSUE:
$ pnpm install --force               # Force reinstall
$ rm -rf node_modules pnpm-lock.yaml # Clean & reinstall

CANNOT CONNECT TO API:
âœ“ Ensure backend is running (check pnpm dev output)
âœ“ Check DATABASE_URL in .env.local
âœ“ Verify PostgreSQL is running
